# YambaAndroid

## How to generate our jacoco test
To make our Test you can execute this following tests
```bash
$ ./gradlew connectedCheck
$ ./gradlew testDebugUnitTest
$ ./gradlew connectedDebugAndroidTest
$ ./gradlew createDebugCoverageReport
$ ./gradlew createDebugCoverageReport
$  ./gradlew test
$ ./gradlew testDebug
$ ./gradlew testRelease
```
